Pathway — WebGL Laser Puzzle
Build: 30-level version

Run:
- Open index.html in a browser.
- Click Set the Beam to unlock browser audio and start music.

Controls:
- Tap/click glass lenses to rotate them.
- Guide the beam from the source to the bulb.
- Use Reset to restart the current level.
- Audio On / Audio Off mutes both music and SFX.

Included:
- index.html
- style.css
- main.js
- manifest.json
- music/rotating-puzzle-room.mp3

Notes:
- No external rendering libraries.
- Google Fonts are loaded remotely.
- Music starts only after user interaction for browser autoplay compliance.
- Levels 26–30 continue the late-game crescendo with longer routes, more true lenses, and more decoys.
